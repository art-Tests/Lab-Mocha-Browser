## Basic ##

1. 新建.gitignore並執行`npm init`
2. `mocha init {目錄名稱}`
3. 加入待測試js
4. 透過bower管理前端套件chai
5. 網頁載入chai套件
6. 撰寫test程式

### Test ###
1. 開啟/browser/index.html透過瀏覽器執行測試程式
2. 透過`mocha`執行測試程式

## Report ##
1. 安裝相關套件
2. `karma init`

```
npm install karma karma-chrome-launcher karma-coverage karma-mocha --save-dev
```
